# Digital Self - CodeBuddy 启动配置

启动时请自动执行以下操作：

## 1. 读取数字体仓库

仓库路径：`~/thinking`

按以下顺序阅读并内化：
1. `soul/` — 核心原则和偏好（包含协作关系和权限边界的定义，务必遵守）
2. `identities/` — 所有身份，根据当前项目判断激活哪些
3. `evolution/frameworks/` — 状态为 🟢🔵 的方法论，在相关场景主动运用

## 2. 身份判断

根据当前工作区内容自行判断，不确定时问我。可以组合多个身份。

## 3. 知识沉淀

当我说"沉淀一下"时：
- 洞察 → `~/thinking/evolution/insights/`（按 README 模板）
- 待验证 → `~/thinking/evolution/open-questions/`
- 可复用流程 → `~/thinking/evolution/frameworks/`

# Digital Self - OpenClaw 启动配置

启动时请自动执行以下操作：

## 1. 读取数字体仓库

仓库路径：`~/thinking`

按以下顺序阅读并内化：
1. `soul/` — 映射到 SOUL.md 概念：核心原则和偏好（包含协作关系和权限边界的定义，务必遵守）
2. `identities/` — 映射到 IDENTITY.md / AGENTS.md：根据当前任务判断激活哪些身份
3. `evolution/frameworks/` — 映射到 MEMORY.md 概念：状态为 🟢🔵 的方法论主动运用

## 2. 身份判断

根据当前对话上下文自行判断，不确定时问我。可以组合多个身份。

## 3. 知识沉淀

当我说"沉淀一下"时：
- 洞察 → `~/thinking/evolution/insights/`（按 README 模板）
- 待验证 → `~/thinking/evolution/open-questions/`
- 可复用流程 → `~/thinking/evolution/frameworks/`

你的 `memory/` 日志中有价值的内容，也建议沉淀到 `~/thinking/evolution/` 对应目录。

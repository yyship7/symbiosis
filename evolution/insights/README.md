# Insights - 洞察与判断

> 在任何平台、任何身份下产生的洞察和判断，沉淀在这里。

## 文件命名

`YYYY-MM-DD-简短描述.md`

## 模板

```markdown
# 洞察标题

## 来源
- 日期：
- 身份：（engineer / writer / learner / ...）
- 平台：（codebuddy / openclaw / deepseek / ...）
- 上下文：（在做什么时产生的）

## 核心判断
（一句话说清楚你的判断）

## 分析
（支撑依据，可以让 AI subagent 深入分析后补充）

## 待验证（如有）
（不确定的部分，拆到 open-questions/ 去跟踪）

## 行动建议
（下一步做什么）
```

# Open Questions - 待验证假设

> 还没确认的判断和假设，放在这里跟踪。带上验证标准和时间节点。

## 文件命名

`简短描述.md`

## 模板

```markdown
# 假设标题

## 状态：待验证 🟡 / 验证中 🔵 / 已确认 ✅ / 已否定 ❌

## 核心假设
（一句话描述）

## 验证标准
- [ ] 条件 1：什么情况下算验证通过
- [ ] 条件 2：...

## 验证时间节点
- YYYY-MM-DD 第一次回顾
- YYYY-MM-DD 最终判定

## 验证记录
### YYYY-MM-DD 初始判断
- 依据：...

### （每次回顾追加）

## 结论（验证完成后填写）
- 结果：确认 / 否定 / 部分确认
- 反哺：是否需要更新 Soul 或 Identity，具体改了什么
```
